## Quick Summaries

### Education & Learning
- **Alpha School**: Multi-age learning platform combining AI and human collaboration for project-based learning

### Lifestyle & Home
- **Homemaid AI**: Low-cost AI substitute for domestic helpers
- **Garden Design AI**: Photo-to-plan garden design with shopping lists and maintenance guides

### Travel & Leisure
- **Getaways AI**: Conversational bot for personalized hiking trail recommendations

### Workplace & Productivity
- **Seen.ai**: Workplace Q&A automation for daily updates and visibility
- **Polymer.ai**: Polyworking platform with canvas builder, user simulation, and business modeling tools
- **Playbooked**: SOP collection system aggregating knowledge from multiple data sources

### Personal Development
- **Book to App**: Convert self-help books into interactive web applications
- **Quote to App**: Transform personality quotes into behavior-modeling web apps

### B2B & Sales
- **LeadIn**: LLM-powered lead discovery through enterprise news analysis

## Repository Structure

Each idea has its own markdown file with:
- Problem/Opportunity description
- Target audience
- Core features
- Technical approach
- Market considerations
- Current status and next steps

## Status Legend
- **Exploring**: Initial research and validation phase
- **Planning**: Defining MVP and technical requirements
- **Building**: Active development
- **Testing**: User testing and feedback
- **Live**: Launched and available
- **Paused**: On hold for now
- **Archived**: No longer pursuing
